When you define add_image_size in function.php file. They will automatically picked up by this plugin and will be displayed in admin dashboard 

== Description ==
If you want to define your own custom image size then use add_image_size() in the function.php file. These image size will be displayed in admin dashboard 
More detail can be found here how to add custom image sizes http://codex.wordpress.org/Function_Reference/add_image_size

== Installation ==

1. Download the plugin and extract the files
2. Upload `admin_dashboard_image_sizes` to your `~/wp-content/plugins/` directory
3. Activate it
4. Go into wp-admin and you will see image size in dashboard
	
Test it out and enjoy!
